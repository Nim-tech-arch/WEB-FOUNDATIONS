# Developer Portfolio v1

A responsive HTML/CSS developer portfolio showcasing personal information, projects, and contact details.

## Features

- **Header** with name and clean typography
- **Navigation bar** with flexbox layout and hover effects
- **About section** with personal description
- **Projects section** displaying projects in a responsive flex layout
- **Contact form** with styled input fields
- **Footer** with copyright information
- **Fully responsive design** with mobile-friendly navigation
- **Modern styling** with clean spacing, centered layout, and professional color scheme

## Technologies Used

- **HTML5** (semantic tags, headings, lists, links, form)
- **CSS3** (Flexbox, responsive design, modern styling)

## How to View

### Option 1: Direct File Opening
Open `index.html` in any modern web browser.

### Option 2: Local Server (Recommended)
Run a local HTTP server for the best experience:

```bash
# Using Python (built-in)
python -m http.server 8000

# Or using Node.js (if available)
npx serve .

# Or using PHP (if available)
php -S localhost:8000
```

Then open `http://localhost:8000` in your browser.
