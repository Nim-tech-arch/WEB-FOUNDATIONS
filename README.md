# Developer Portfolio v1

A simple HTML-based developer portfolio showcasing personal information, projects, and contact details.

## Features

- Header with name
- Navigation bar
- About section
- Projects section (listing 4 Python projects)
- Contact form
- Footer

## Technologies Used

- HTML5 (semantic tags, headings, lists, links, form)

## How to View

Open `index.html` in any web browser.
